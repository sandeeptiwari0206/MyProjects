3Ô∏è‚É£ Install MongoDB 6.0
-------------------------
# Import GPG key
curl -fsSL https://www.mongodb.org/static/pgp/server-6.0.asc | sudo gpg --dearmor -o /usr/share/keyrings/mongodb-server-6.0.gpg

# Add MongoDB repository
echo "deb [ arch=amd64 signed-by=/usr/share/keyrings/mongodb-server-6.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/6.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-6.0.list

# Install MongoDB
sudo apt update
sudo apt install -y mongodb-org

# Start and enable MongoDB
sudo systemctl enable mongod
sudo systemctl start mongod

4Ô∏è‚É£ Secure MongoDB
--------------------
Edit MongoDB Config:
sudo nano /etc/mongod.conf

Ensure:
net:
  bindIp: 127.0.0.1

Add under the file:
security:
  authorization: enabled

Restart MongoDB:
sudo systemctl restart mongod

5Ô∏è‚É£ Create MongoDB Users
--------------------------

Admin User:
mongosh
use admin
db.createUser({
  user: "admin",
  pwd: "StrongPasswordHere123!",
  roles: [{ role: "userAdminAnyDatabase", db: "admin" }]
})
exit

Reconnect:
mongosh -u admin -p StrongPasswordHere123! --authenticationDatabase admin

App DB & User:
use game_db
db.createUser({
  user: "gameuser",
  pwd: "GameUserPassword@123",
  roles: [{ role: "readWrite", db: "game_db" }]
})
exit

7Ô∏è‚É£ Configure Environment Variables

8Ô∏è‚É£ Start Node App Using PM2
-----------------------------

Copy and run the command shown after `pm2 startup`.

Logs:
